Test Node

¡Bienvenido!

A continuación se describen las actividades a realizar para la prueba de desarrollador Node. Por favor ten en cuenta los siguientes puntos.
  • Eres libre de realizar cualquier consulta.
  • Se evaluará estructura de código y lógica implementada.
  • El ejercicio debe ser subido en un repositorio github.

Actividades principales

  • Realizar formulario de inicio de sesion.
  • los datos del formulario deben validarse con la información disponiblle en el archivo ./data.json:
    • Si el usuario no existe, debe mostrarse una pantalla de error.
    • Si el usuario existe, debe ir a la pagina de opciones.
  • Si el usuario inicia sesión, debe poder realizar las siguientes acciones:
    • Ver imagenes.
    • Cargar Imagenes.
    • Crear usuarios (Sólo si es admin).

Carga de imagenes

  • Las imagenes deben cargarse en una carpeta con el nombre del usuario.
  • El usuario debe poder eliminar las imagenes
  • Un usuario solo puede ver y eliminar sus imagenes

Administrador

Las acciones del administrador son las siguientes:

  • Crear usuarios
  • Eliminar usuarios
  • Cargar imagenes
La edición de usuarios debe modificar el archivo ./data.json
